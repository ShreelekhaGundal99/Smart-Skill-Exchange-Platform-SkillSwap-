# 🚀 SkillSwap

> **Learn. Teach. Grow Together.**

SkillSwap is a responsive frontend web application that connects students who want to exchange skills and learn collaboratively. Instead of learning alone, students can discover peers with complementary skills, build meaningful connections, and grow together through knowledge sharing.

---

## 📖 Project Overview

SkillSwap provides a simple and engaging platform where students can:

- 🔍 Discover skilled peers
- 🤝 Find compatible learning partners
- 👨‍🏫 Teach their expertise
- 📚 Learn new technologies
- 🏆 Earn recognition through a gamified leaderboard

The project is designed with a modern, colorful, and interactive user interface using only **HTML, CSS, and JavaScript**.
